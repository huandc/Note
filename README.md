# 段落格式
Markdown 段落没有特殊的格式，直接编写文字就好，段落的换行是使用两个以上空格加上回车  
```你好<kbd>空格</kbd><kbd>空格</kbd>  
我在这里。<kbd>空格</kbd><kbd>空格</kbd>  ```

你好  
我在这里

# 删除线
如果段落上的文字要添加删除线，只需要在文字的两端加上两个波浪线 ~~ 即可，实例如下：  
```
RUNOOB.COM  
GOOGLE.COM  
~~BAIDU.COM~~  
```
~~BAIDU.COM~~
